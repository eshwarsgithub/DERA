## Summary

1. **User visits** `/des` page
2. **Next.js calls** `/api/de/inventory`  
3. **API route calls** `fetchDataExtensions()` from sfmcClient
4. **Client checks** if SFMC credentials configured
5. **If yes**: Requests OAuth token from SFMC
6. **Uses token** to fetch Data Extensions via SOAP
7. **Retrieves** field details for each DE
8. **Scans** each field for PII indicators
9. **Computes** risk score based on PII + usage
10. **Returns** enriched data to frontend
11. **UI renders** table with risk scores, PII tags, and action buttons
